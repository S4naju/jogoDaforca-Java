# jogo da Forca em Java

### Descrição do projeto:
  Este projeto foi desenvolvido durante o meu ensino médio integrado ao curso técnico de informática no IFRJ – Campus Arraial do Cabo, como parte da disciplina de Programação I.
  
  O objetivo do trabalho foi criar um jogo em Java aplicando os conteúdos aprendidos em sala de aula, como estruturas de repetição, condicionais e manipulação de strings.

---

### Funcionamento do Jogo

O Jogo da Forca roda no terminal e desafia o jogador a descobrir uma palavra secreta com base em uma dica.
O menu permite acessar:
- Sobre o jogo
- Jogar
- Créditos
- Avaliação
- Sair

Durante o jogo, o jogador escolhe um número de 1 a 6, cada um com uma palavra diferente, e tem 12 tentativas para acertar.
O sistema mostra as letras corretas, o número de tentativas restantes e exibe mensagens de vitória ou derrota.

---

### Como visualizar 

Para jogar, basta executar o arquivo Java correspondente no seu ambiente de desenvolvimento (como NetBeans, Eclipse ou VS Code com suporte a Java).
